# Trackpad e Mouse
- **scorrimento**: su un trackpad lo scorrimento avviene con 2 dita
- **Cliccare**: il trackpad è un unico grande pulsante, si può cliccare dappertutto
- Clic con il tasto destro o con due dita: Per aprire il menu contestuale "clic destro", posizionare due dita sul trackpad e fare "clic" con una di esse. È anche possibile fare clic con il tasto destro del mouse o tenere premuto il tasto Control e fare clic.
# Shortcut della tastiera
- Copiare: Comando + C
- Incolla: Comando + V
- Sposta (Taglia): Comando + Opzione + V
- Annulla: Comando + Z
- Seleziona tutto: Comando + A
- Cambia app/finestra: Comando + Tab
- Ridurre a icona l'app/finestra: Comando + M
- Chiudere un'applicazione: Comando + Q
- Chiudere la finestra/tab: Comando + W
- Effettuare uno screenshot (dell'intero schermo): Maiusc + Comando + 3
# Dock
Nella parte inferiore dello schermo si trova il Dock di macOS. È l'equivalente Mac della barra delle applicazioni di Windows. È uno dei modi più semplici per avviare e accedere alle applicazioni. Nel Dock sono presenti due aree separate da una partizione. A sinistra si trovano le applicazioni e a destra le cartelle, il Cestino e le finestre aperte ridotte a icona.
![](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2019/08/dock.png?q=50&fit=crop&w=480&dpr=1.5)
Per appuntare un elemento nel Dock, si fa clic con il pulsante destro del mouse (o con due dita su un trackpad), quindi si sceglie Opzioni > Mantieni nel Dock. Per eliminare un elemento, fare clic e trascinarlo finché non appare "Rimuovi", quindi rilasciarlo. È possibile configurare il Dock in modo che appaia in basso o sul bordo destro o sinistro dello schermo. È anche possibile configurare il Dock in modo che si nasconda automaticamente. Avviare Preferenze di sistema > Dock per configurarlo come si desidera.
# Barra di menu
Nella parte superiore dello schermo si trova la barra dei menu del Mac (illustrata di seguito). A differenza di Windows, dove i menu a discesa come File e Modifica appaiono agganciati alla finestra in uso, macOS li colloca sempre nella parte superiore dello schermo. È possibile sapere quale applicazione è in uso perché il suo nome appare nell'angolo in alto a sinistra accanto al logo Apple.
![](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2019/08/menu_bar.png?q=50&fit=crop&w=480&dpr=1.5)
A destra della barra dei menu si trova l'equivalente di Apple della barra delle applicazioni di Windows (illustrata sotto). È qui che si eseguono operazioni come la connessione alle reti Wi-Fi o il controllo della percentuale di batteria. Molte applicazioni, come Evernote e Google Drive, posizionano qui le icone per facilitarne l'accesso. macOS ha anche un buon numero di app che si trovano nella barra dei menu.
![](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2019/08/systray.png?q=50&fit=crop&w=480&dpr=1.5)
# Spotlight
Il Dock è uno dei modi più semplici per accedere alle applicazioni, ma non è il più efficiente. Premendo Comando + Barra spaziatrice, si avvia la ricerca Spotlight. Si tratta del motore di ricerca onnicomprensivo del Mac ed è il modo perfetto per avviare le applicazioni: basta digitare il nome dell'applicazione e premere Invio.
![](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2019/08/spotlight.png?q=50&fit=crop&w=480&dpr=1.5)
Con Spotlight si possono fare molte cose. È possibile accedere ai pannelli delle opzioni nelle Preferenze di Sistema, cercare file e persino eseguire semplici somme o convertire valuta. È anche possibile utilizzare il linguaggio naturale nella ricerca, ad esempio "file PDF che ho aperto la settimana scorsa", per affinare ulteriormente i risultati. È una buona idea prendere l'abitudine di usare Spotlight, in particolare per lanciare le applicazioni.
# Preferenze di sistema
L'equivalente Mac del Pannello di controllo di Windows è Preferenze di sistema. Qui è possibile aggiungere nuovi utenti al computer, modificare le impostazioni di sicurezza o personalizzare il desktop (solo per citare alcune delle sue utili funzioni). Anche le applicazioni di terze parti possono installare qui i propri pannelli di opzioni. Vale la pena di curiosare nelle Preferenze di Sistema per familiarizzare con le varie opzioni.
![](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2019/08/system_preferences.png?q=50&fit=crop&w=480&dpr=1.5)
# Installa e disinstalla applicazioni
## installa
Il processo di installazione di un software su Mac è leggermente diverso da quello di un computer Windows, ma è comunque semplice. Esistono due metodi principali per installare un software su Mac:
- Programma di installazione dei pacchetti: Funzionano come le procedure guidate di installazione di Windows. Fate doppio clic sul file PKG per eseguirlo. Seguite le istruzioni sullo schermo (di solito basta fare clic su "Avanti" un paio di volte) finché il software non viene installato.
- Installazioni su Mac App Store: Avviare il Mac App Store e trovare l'applicazione che si desidera scaricare. Fate clic su "Ottieni" (o "Acquista" se si tratta di un'applicazione a pagamento) e digitate la password del vostro ID Apple. L'applicazione si installa automaticamente nella cartella Applicazioni.
![](https://static1.howtogeekimages.com/wordpress/wp-content/uploads/2019/08/mac_app_store.png?q=50&fit=crop&w=480&dpr=1.5)
## disintalla
I due metodi principali per rimuovere il software sono:
- Eliminazione manuale: Individuare l'applicazione nella cartella Applicazioni, quindi fare clic e trascinarla nel Cestino. Potrebbe essere necessario fornire la password di amministratore per disinstallare completamente un'applicazione. Svuotare il Cestino per recuperare lo spazio libero.
- Disinstallatori automatici: Alcune applicazioni includono disinstallatori che funzionano come quelli di Windows, quindi controllate prima la cartella Applicazioni. Se si trova un programma di disinstallazione per un'applicazione, fare doppio clic su di esso e seguire le istruzioni visualizzate.
