## Distanza percorsa
$$\text{La distanza percorsa è la lunghezza complessiva del tragitto}$$
## Spostamento
$$\text{Lo spostamento }\Delta_x \text{di un punto materiale è il cambiamento di posizione, cioè la differenza fra la posizione finale } X_{f} \text{e quella iniziale} X_{i} $$
$$\Delta_{x} = x_f-x_i$$
## Velocità scalare media
$$\text{Velocità scalare media}= \frac{\text{distanza persocrsa}}{\text{tempo impiegato}}$$
## Velocità media
$$\text{La velocità media }V_{m} \text{ è il rapporto tra lo spostamento } \Delta_{x} \text{ e il tempo impiegato } \Delta_{t} $$
## Velocità istantanea
$${lim\ di\ t\rightarrow0}\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \frac{\Delta_x}{\Delta_t}$$
## Moto rettilineo uniforme (MRU)
$$\text{rettilineo con V=K(=costante)}$$
## Legge oraria MRU
$$\begin{cases} V=K\\X=X_0+V*t \end{cases}$$
## Accellerazione
$$\begin{align*}
		\text{Velocità=variazione della posizione}\\
		\text{Accellerazione=variazione della velocità}
\end{align*}$$
## Accellerazione media
$$a_m=\frac{\Delta_V}{\Delta_t}=\frac{V_f-V_i}{t_f-t_i}[m/s^2]$$
## Velocità istantanea
$$a_i=lim\ \Delta t \rightarrow 0 \ \ \ \ \ \ \ \ \frac{\Delta V}{\Delta t}$$
## Legge oraria MUA
$$\begin{cases} a=K\\V=V_0+a*t\\X=X_0+(V_0+\frac{1}{2}a*t)t \end{cases}$$
## Gravità 
$$\begin{align*}
		\text{Gravità Terra=g}\\
		g=9,81\frac{m}{s^2}
\end{align*}$$
